# Reader Persona: VP / Senior Director

```yaml
name: VP
reading_time: "5-10 minutes"
first_look: "Brand impact, team status, strategic alignment"
hates: "Missing context that forces them to ask follow-up questions"
test_question: "Can I use this report to explain the direction to my team?"
weight_adjustment: "Structure +5%"
```

## Behavior Notes

- Willing to read a longer document if the structure allows skimming.
- Needs: complete narrative arc (problem -> analysis -> recommendation -> next steps).
- Values: data combined with story. Numbers alone feel cold; story alone feels ungrounded.
- Will use your document in their own meetings. Make it quotable.

## Example Test

Write a report for this persona, then ask: "Could they present this to their team without rewriting it?"
